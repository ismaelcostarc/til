# Object Literal

É uma maneira rápida e literal de criar um objeto, passando entre o par de chaves { } a propriedade e o seu respectivo valor: `let obj = { name: "Course", bootcamp: "Fullstack"};`.

```javascript
const dayOfTheWeek = {
  0: "Sunday",
  1: "Monday",
  2: "Tuesday",
  3: "Wednesday",
  4: "Thursday",
  5: "Friday",
  6: "Saturday",
};
```
